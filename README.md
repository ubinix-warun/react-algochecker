# Config & Build

* Edit Tatum API key on src/config.json
```
{
    "TATUM_API_KEY": "<TATUM_API_KEY>",
    "TATUM_API_URL": "https://api-eu1.tatum.io"
}
```
* Run build script with npm
```
nvm use v16.14.0
npm run install

npm run build
```
In the project directory, you can run:

### `yarn start`
### `yarn test`

### `yarn build`
